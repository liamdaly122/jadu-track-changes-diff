    Install the Jadu bookmarklet

    One click on any Jadu Track Changes page opens this tool in a new tab with the page already compared. Saves you copying and pasting the source HTML each time.

    Install (once)

    1. Show your browser's bookmarks bar (Cmd+Shift+B on Mac, Ctrl+Shift+B on Windows).
    2. Drag the blue button below up onto your bookmarks bar.

    Open in Jadu Diff

    Use it

    1. Open any Track Changes page in Jadu.
    2. Click the "Open in Jadu Diff" bookmark.
    3. This tool opens in a new tab with the page already compared.
